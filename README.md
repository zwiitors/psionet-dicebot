# ΨI／ONET ダイスボット

# 1. 概要
zwiitors作のTRPG『ΨI／ONET』『シノギ』などの判定システムで使用するためのDiscord用ダイスボットです。

# 2. 使い方
## 通常の判定
失敗するか、達成度が5になるまでd100の判定を行う。判定が終了すると、その判定の達成度が表示される。また、その判定で振り直しや修正を行っていた場合、消費した士気の合計値が表示される。
```
/roll 成功率
```
### 振り直し
達成度が5に満たない状態で判定が失敗したとき、以下のコマンドで判定を続きから振り直す。
```
r
```
### 修正
達成度が5に満たない状態で判定が失敗したとき、以下のコマンドで判定の出目を修正する。修正値は、次の達成度に達するまで自動的に補正される。
```
m
```
### 終了
達成度が5に満たない状態で判定が失敗したとき、以下のコマンドで判定を終了する。
```
f
```
### 複数回の判定
同じ成功率の判定を複数回繰り返す場合、以下のコマンドで判定する。
```
/roll 成功率x回数
```
### 達成度補正付きの判定
判定に達成度の補正（例：達成度+1）がついている場合、以下のコマンドで判定する。
```
/roll 成功率 達成度補正
```
判定を複数回行う場合、以下のように達成度の補正を指定する。達成度補正の数が回数に満たない場合、1回目の判定から順に補正が適用される。
```
/roll 成功率x回数 達成度補正,達成度補正,達成度補正,...
```
## ダメージダイス
ダメージを計算する。通常のダイス機能としても使える。ダイス式には、1d10のような基本的な形式から、1d10+2d4+4のような複数のダイス・固定値を含むものまで使用可能。ダメージダイスの結果が表示されるとき、出目の合計値と期待値が表示される。
```
/damage ダイス式
```
### 達成度倍率込みのダメージダイス
ΨI／ONETにおいて、攻撃・防御の達成度に応じたダメージ・防御力の倍率を計算したい場合は、以下のコマンドを使用する。
```
/damage ダイス式 達成度
```
### 達成度倍率込みのダメージダイス（複数回）
ΨI／ONETにおいて、攻撃・防御の達成度に応じたダメージ・防御力の倍率を計算したい場合は、以下のコマンドを使用する。入力した達成度の数が判定の回数になる。
```
/damage ダイス式 達成度,達成度,...
```
## 振り直しコストの変更
デフォルトの振り直しコストは7となっている。これを変更する場合、以下のコマンドを使用する。このコストはユーザーごとに保存される。
```
/r_cost コスト
```
